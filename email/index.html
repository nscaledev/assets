Email Signature Generator

Instructions

  1. Enter your details in the form below
  2. Double check the preview
  3. Click the button to copy the signature to the clipboard
  4. In Gmail, click on Settings in the top right
  5. Select See all settings
  6. Under Signature, Create new and name it nscale, then paste (e.g. ctl + V) into the textbox
  7. Select nscale as the default for new emails
  8. Select Insert signature before the quoted text in replies
  9. Click Save changes down the bottom
